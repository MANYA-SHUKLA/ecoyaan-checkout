# Ecoyaan Checkout

A checkout flow for Ecoyaan — cart, shipping/address, and payment. Built with Next.js so it’s fast and easy to work on.

## What’s in here

- **Cart** — View and tweak items, apply coupons.
- **Checkout** — Shipping address form with pincode lookup.
- **Payment** — Card-style payment step (UI only; no real payments).
- **Success** — Order confirmation with a bit of confetti.

## Tech

- Next.js 14, React, TypeScript
- React Hook Form + Zod for forms and validation
- Framer Motion for animations, Tailwind for styling
- React Hot Toast for notifications

## Run it
ash
npm install
npm run dev
